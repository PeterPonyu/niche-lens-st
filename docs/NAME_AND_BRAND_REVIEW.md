# Name and brand review

Date: 2026-05-27

## Candidate name

**NicheLens-ST**

## Suitability

- Communicates the project focus: viewing and modeling cellular niches in spatial transcriptomics.
- Distinct from the primary baseline name CellNiche while still being semantically legible.
- Suffix `-ST` clarifies spatial-transcriptomics scope.
- GitHub search for `NicheLens-ST`, `NicheLensST`, and `niche-lens-st` returned no exact repository-name collision during the 2026-05-27 check.
- Web search for exact quoted variants returned no obvious exact-name paper/tool collision during the 2026-05-27 check.

## Branding boundary

NicheLens-ST should not use CellNiche/scComm naming patterns for modules, figures, or claims. Baselines are cited as prior art and comparison targets only.

## Decision

Keep `NicheLens-ST` / repository folder `niche-lens-st` for now. Re-check before public release or GitHub remote creation.
